f) Zoek op of experimenteer wat += doet. Werkt deze ook met Strings? Weet je hoe je deze anders kan schrijven?

Met += doe je niet alleen de berekening, maar wijs je ook gelijk de waarde toe aan de variable. Je kunt dit inderdaad met strings ook gebruiken. Je kan dit ook schrijven door te concatenaten met string + string. Je kan dit alleen niet met strings en -= doen, dan geeft ie aan dat het NaN is, Not a Number.
