Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

De values voor de position property zijn: static, relative, absolute, fixed en sticky

De initial value is static. En betekent dat het element gewoon op de plaats zit waar die hoort.

relative geeft een off-set op de plek waar het element oorspronkelijk hoort te zitten.

absolute haalt het element uit de normale flow van het document en plaatst deze bij het dichtstbijzijnde positioned ancestor. (ancestor die niet static is) Als deze niet bestaat komt ie bij het ICB (initial containing block) die het containing block van het document's root element is

fixed positioning lijkt op absolute alleen is het elements containing block de initial containing block dat is vasgesteld door de viewport.

Sticky positioning kun je zien als een hybrid tussen relative en fixed positioning. Deze gedraagt zich als relative tot hij een bepaalde grens overschrijdt, dan gedraagt hij zich als fixed tot het de grens van zijn parent bereikt.
