Styling die je toepast wordt toegepast in een bepaalde volgorde. Het kan voorkomen dat je styling toepast en vervolgens niet het gewenste resultaat ziet. Cascade en Inheritance zijn belangrijke concepten om te begrijpen hoe styling werkt. Lees hier in de officiÃ«le documentatie meer over cascade en inheritance. Zoek ook andere referenties om het concept te begrijpen.

In eigen woorden wordt met cascade bedoelt dat de rules van boven naar beneden gaan. Dus als je ergens bovenin je css de color van alle h1's rood maakt, en vervolgens ergens onderin de css de h1 groen maakt, en de specificity van de selectors is even groot, dan is het eindresultaat groen.

Inheritance wilt zeggen dat sommige properties door de descendants wordt overgenomen van de parent. Een voorbeeld van een property die inherit wordt van de parent is color.

De gebruikte code is:

div { color: white;} div { color: red;}
