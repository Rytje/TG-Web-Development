Er zijn verschillende selectors, bijvoorbeeld; type selectors descendant selectors class selectors Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

De type selector matched elementen door middel van node name. Het selecteert dus alle elementen van het gegeven type binnen het document. Het voordeel hiervan is dat je een groot aantal elementen in een keer kan selecteren.

Descendant selector matched alle elementen die descendants zijn van een specifieke element. Het voordeel hiervan is dat je makkelijk elementen kan selecteren die binnen een ander element zitten en niet uit maakt hoe diep de descendant zit.

De class selector selecteert elementen die bij een bepaalde class attribute horen. Het voordeel hiervan is dat je een rule kan maken voor bijvoorbeeld alle p elementen kan selecteren, maar alleen die bij een bepaalde class horen.
