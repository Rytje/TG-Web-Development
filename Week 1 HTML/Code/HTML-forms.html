Vraag 10

<table>

Dit is het element dat alle overige elementen overkapt dat met table te maken heeft. Dit gebruik je als je een 2-d table wilt maken dat bestaat uit rows en columns waarin cells zitten die data kunnen bevatten.

<thead>

Dit element is het overkoepelende element voor de th elementen, dit is de ruimte voor de headers van de table.

<tbody>

Net als de html body, is dit de ruimte voor de werkelijke inhoud van de table. Hierin gaat de tr elementen.

<tfoot>

Hierin gaat het voetstuk van de table, dit kan bijvoorbeeld zijn het totaal bedrag als het gaat om een factuur etc.

<th tr td>

De th definieert een cell als een header va een groep cellen in een table. De tr element definieert een row cells in een table. Het td element difinieert een cell van een table dat data bevat.
