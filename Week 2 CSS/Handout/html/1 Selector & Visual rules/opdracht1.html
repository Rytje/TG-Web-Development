Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Je kunt CSS op 3 manieren in HTML gebruiken. Inline, intern en extern. Inline betekent dat je het in de de css toevoegt aan de html tags in het style attribute. Intern gaat de css in de head van het html document binnen het style element. Extern betekent dat de css in een apart css bestand zit, en in de head gelinkt wordt aan het html document door middel van een referentie in een link element.
